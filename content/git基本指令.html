git基本指令

git add (註冊檔案或目錄到索引)

git commit (提交添加到索引的檔案)

git status (顯示修改檔案清單)

git log (顯示提交記錄)

git rm <file> (刪除檔案)

git show <commit> (查看提交的詳細記錄)

git clean (從工作目錄刪除非管理對象的檔案)

git checkout -- <file> (還原在工作目錄已更改的檔案)

git add -u (只註冊已提交過的檔案到索引)

git branch (列出所有 branch)

git pull (更新遠端回來)

git push (送檔案到遠端)

git merge (合併檔案)
